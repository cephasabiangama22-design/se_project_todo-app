# Simple Todo App

A simple and interactive Todo application that allows users to create, organize, and manage daily tasks. Users can add new tasks, mark them as completed, delete tasks they no longer need, and assign due dates to help stay organized.

## Functionality

The Simple Todo App provides an easy way to keep track of tasks through a clean and responsive interface. Users can:

Add new todo items using a form.
Mark tasks as completed or incomplete with a checkbox.
Delete tasks from the list.
Display and format due dates for each task.
Validate form inputs before allowing submission.
Prevent invalid or empty tasks from being added.
Receive real-time validation feedback while typing.

## Technology

This project was built using the following technologies and techniques:

HTML5 for the application's structure.
CSS3 for styling and responsive layout.
JavaScript (ES6+) for application logic and interactivity.
Object-Oriented Programming (OOP) using JavaScript classes to organize the code.
DOM Manipulation to dynamically create, update, and remove todo items.
Form Validation using a custom FormValidator class to validate user input and control the submit button.
Event Listeners to handle user interactions such as adding, deleting, and completing tasks.
Git for version control.
GitHub for source code hosting.
GitHub Pages for deployment
